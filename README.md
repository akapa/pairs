# Pairs Game

## Installation

```bash
npm install
```

## Usage

```bash
npm run start
```

## Development

### Watch to build

Watches Javascript code to transpile with Babel and SASS to generate CSS.

```bash
npm run watch
```

### Build once

Transpiles JS and generates SASS from CSS once.

```bash
npm run build
```

### Lint code

```bash
npm run lint
```

### Run HTTP server

```bash
npm run serve
```
